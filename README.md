# BasicClay - Learning Spring Boot CRUD Operations

## Introduction

Welcome to BasicClay! This project is designed to help team members learn and practice basic Spring Boot annotations and CRUD (Create, Read, Update, Delete) operations. By working with this project, you'll gain hands-on experience in building a simple Spring Boot application with a focus on database operations.
